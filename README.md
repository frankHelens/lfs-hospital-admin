# 前端编码规范：
* 好的代码是不需要解释的~~~
## 命名
1. 命名尽量语义化，全拼
## js:
1. js变量统一使用小驼峰命名法(itemName)
2. js使用函数式编程(map,filter,reduce,find,findIndex,forEach)
3. 不要使用for循环
## css:
1. css变量全部使用横杠(item-name)
2. css统一使用flex布局，不要用使用float
3. 尽量使用封装好的scss函数
## pug:
1. 没有属性不要写空括号
